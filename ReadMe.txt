Приложение для просмотра фотографий ВКонтакте
1. Можно авторизоваться и просмотреть свои альбомы и фотографии
2. После авторизации приложение "запомнит" вас, не надо будет постоянно авторизовываться
3. Есть возможность выйти из аккаунта, потом нужно будет заново авторизоваться
4. Есть четыре экрана: авторизация, экран с альбомами, экран с фотографиями выбранного альбома, экран с фотографией во весь экран

Стек:
1. Clean architecture, MVVM
2. VK api
3. Di Koin
4. Navigation Architecture Component with safe args
5. RxJava
6. Retrofit
7. Picasso
